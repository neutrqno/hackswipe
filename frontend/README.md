# HackSwipe 🚀

## Features

- Create a hacker profile
- Add skills and hackathon you're attending
- Swipe through potential teammates
- Filter by hackathon name
- Copy GitHub / LinkedIn / Discord to contact teammates
- 

## Tech Stack

Frontend:
- Next.js
- React
- TailwindCSS

Backend:
- Node.js
- Express

Deployment:
- Vercel (frontend)
- Render (backend)
